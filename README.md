# Career_Spark

Career_Spark is an AI-powered multi-agent career advisor system that provides personalized career advice with fact-checking and persistent memory. It leverages multiple agents collaborating to deliver accurate and insightful guidance.

## Features
- Multi-agent collaboration (CareerAdvisor, FactChecker)
- Real-time Google Search integration
- Persistent session memory
- CLI interface for user interaction

## Setup
1. Clone the repository:  
   `git clone https://github.com/zahi-2005/Career_Spark.git`
2. Navigate to the project folder:  
   `cd Career_Spark`
3. Install dependencies:  
   `pip install -r requirements.txt`  
   *(create the requirements.txt if not already available)*
4. Add your API keys in `config.py`
